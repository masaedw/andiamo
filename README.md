# Andiamo! — 会話で覚えるイタリア語

対訳付き会話を軸に、文法を段階的に学ぶイタリア語学習アプリ。全8課。
ビルド不要の静的 Web アプリ（vanilla HTML/CSS/JS）。

## 起動

```bash
cd andiamo
python3 -m http.server 8642
# → http://localhost:8642/ を開く
```

`index.html` をブラウザで直接開いても動く（Google Fonts の読み込みのみネットワークが必要）。

## 機能

- **会話（Dialogo）**: 二人のやりとりのリレー形式・対訳付き。行ごと再生／全体再生（再生中の行をハイライト）／日本語訳を隠す練習モード／再生速度切替
- **登場人物の個性**: 各話者に話し方・口癖の設定があり（例: マルコは Senti / Dai、通行人は Si figuri!）、会話文と画面のキャスト紹介に反映。音声も人物ごとに別の声を割り当てる（性別に合う OS 音声を選択、主人公 Yui は全課で同じ声に固定。声が足りない場合はピッチ差で区別）
- **文法（Grammatica）**: その課の会話に登場する文法の解説。活用表・例文すべてに音声ボタン
- **単語（Parole）**: 一覧表示・フラッシュカード（シャッフル・自動読み上げ）・全単語連続再生
- **クイズ（Quiz）**: 単語から8問を自動出題（伊→日／日→伊の4択）。80% 以上で課の完了。自己ベストを記録
- 進捗は localStorage に保存

## 音声

Web Speech API（`speechSynthesis`）の it-IT 音声を使用。音声ファイル不要。
イタリア語音声が OS にない場合は画面に警告を表示する（macOS: 設定 → アクセシビリティ → 読み上げコンテンツ → 声を管理 からイタリア語を追加）。

## カリキュラム

| 課 | タイトル | 場面 | 文法 |
|----|---------|------|------|
| 01 | Piacere! | 語学学校で初対面の挨拶 | essere・否定文・挨拶 |
| 02 | Al mercato | 市場で果物を買う | 名詞の性・複数形・冠詞・avere |
| 03 | Le cartoline | 土産物店で絵葉書を買い、出身地を話す | -are 動詞・出身表現・敬称 Lei |
| 04 | Alla stazione | 駅で切符を買う | -ere / -ire 動詞・時刻・数字 |
| 05 | In trattoria | レストランで好みを伝えおすすめを聞く | piacere 構文・食事の流れ |
| 06 | In albergo | ホテルでチェックイン | volere / potere / dovere |
| 07 | Per strada | 道を尋ねる | 前置詞・冠詞前置詞・位置表現 |
| 08 | Il weekend | 週末の思い出を話す | 近過去（passato prossimo） |

## 構成

```
index.html      — エントリポイント
css/style.css   — スタイル
js/data.js      — 全8課のコンテンツ（会話・文法・単語）
js/app.js       — ルーティング / TTS / 描画 / クイズ / 進捗
```

課の追加は `js/data.js` の `UNITS` 配列にオブジェクトを足すだけでよい。
話者には `gender`（声の割当に使用）と `style`（話し方・口癖。キャスト紹介に表示）を設定する。
